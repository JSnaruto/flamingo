# Heading Component

## Usage

```jsx
import { Heading } from '@heetch/flamingo-react';

export default () => <Heading>Content</Heading>;
```

## Props

#### `as`: oneOf(`HEADING_LEVELS`)

```jsx
import { Heading } from '@heetch/flamingo-react';

export default () => <Heading level={3} />;
```

#### `className`: String

```jsx
export default () => <Heading className='myClass' />;
```
